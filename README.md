# Micro-Simulação Social com Agentes Generativos

Recriação simplificada da arquitetura do paper *Generative Agents: Interactive
Simulacra of Human Behavior* (Park et al., UIST 2023).

Cinco agentes com persona, memória, reflexão e planejamento convivem numa vila
textual por 3 dias.
